# 宁听 (ning-ting) 需求文档

- **版本**：v1.0.0
- **最后更新**：2026-04-14
- **产品定位**：纯前端、无后台的小学生专属听写助手

---

## 一、文档说明

本文档为「宁听」项目的详细需求规格说明书，涵盖功能需求、非功能需求、数据结构、UI 设计规范和验收标准，供开发实施参考。

## 二、产品概述

### 2.1 产品愿景

为小学生（尤其是宁宁）提供一个**纯净、无广告、数据自主掌控**的听写练习工具。无需注册登录，打开即用，所有数据存储在浏览器本地，支持导入导出备份。

### 2.2 目标用户

| 用户角色 | 核心诉求 |
|---------|---------|
| 小学生（主要使用者） | 自主完成听写练习，操作简单，界面友好 |
| 家长（辅助者） | 拍照批改、查看正确率、管理词库、数据备份 |

### 2.3 核心价值

- **完全本地化**：无服务器，无数据上传，隐私安全
- **多类型词库**：支持汉字、词语、英语单词、古诗
- **灵活听写**：语音朗读可调速、可设置重复次数
- **人工批改**：拍照对照，手动标记对错，家长参与
- **数据可迁移**：一键导入/导出 JSON 备份文件

## 四、非功能需求

### 4.1 性能要求

| 指标 | 目标值 |
|------|--------|
| 首屏加载 | < 2 秒 |
| 页面切换 | < 300ms |
| 语音朗读响应 | 点击后 < 500ms 开始播放 |
| 数据存储 | 支持 5000+ 词条流畅操作 |
| 照片拍摄 | 压缩后存储，单张 < 200KB |

### 4.2 兼容性要求

| 平台 | 最低版本 |
|------|----------|
| Chrome | 90+ |
| Edge | 90+ |
| Safari | 14+ |
| iOS Safari | 14+ |
| Android Chrome | 90+ |

### 4.3 可用性要求

- 所有可点击元素触控区域 ≥ 44x44px（移动端友好）
- 关键操作提供二次确认（删除、清空数据）
- 语音朗读失败时给出友好提示（如「浏览器不支持语音朗读」）
- 首次使用展示功能引导浮层

### 4.4 安全性/隐私要求

- 所有数据仅存储在用户浏览器 localStorage
- 拍摄的照片仅存内存（blob:xxx），不上传任何服务器
- 不收集任何用户个人信息
- 隐私声明明确告知「数据完全本地化」

## 五、数据结构设计

### 5.1 顶层存储结构

```typescript
interface AppData {
  version: string;           // 数据版本号，如 "1.0.0"
  textbooks: Textbook[];
  lessons: Lesson[];
  words: WordItem[];
  records: DictationRecord[];
  settings: AppSettings;
}
```

### 5.2 各实体定义

```typescript
interface Textbook {
  id: string;                // nanoid
  name: string;              // 如 "部编版语文一年级上册"
  subject: 'chinese' | 'english' | 'poetry';
  grade: 1 | 2 | 3 | 4 | 5 | 6;
  term: 1 | 2;               // 1=上册, 2=下册
  coverColor: string;        // 封面主色，如 "#FF9A62"
  createdAt: number;         // 时间戳
  updatedAt: number;
}

interface Lesson {
  id: string;
  textbookId: string;
  name: string;              // 如 "第1课 秋天"
  order: number;             // 排序序号
  createdAt: number;
  updatedAt: number;
}

interface WordItem {
  id: string;
  lessonId: string;
  type: 'hanzi' | 'ciyu' | 'english' | 'poetry';
  content: string;           // 听写朗读的核心内容
  pronunciation?: string;    // 拼音/音标
  hint?: string;             // 提示语
  extra?: {                  // 扩展字段
    author?: string;         // 古诗作者
    fullText?: string;       // 古诗全文
    example?: string;        // 英语例句
  };
  order: number;
  createdAt: number;
  updatedAt: number;
}

interface DictationRecord {
  id: string;
  textbookId: string;
  lessonId: string;
  startTime: number;
  endTime: number;
  itemIds: string[];         // 本次听写的词条ID列表（按顺序）
  results: Record<string, boolean>;  // { [itemId]: true=对, false=错 }
  accuracy: number;          // 0-100
  photoUrl?: string;         // 拍照的 blob URL（可选，仅内存）
}

interface AppSettings {
  defaultSpeechRate: number;      // 0.5-1.5，默认 1.0
  defaultRepeatTimes: 1 | 2 | 3;  // 默认 2
  theme: 'light' | 'auto';        // 默认 auto
}
```

## 六、UI 设计规范

### 6.1 设计原则

- **圆润**：所有卡片、按钮使用大圆角（16px - 32px）
- **轻透**：毛玻璃效果，半透明背景，浅投影
- **温暖**：暖白底色 + 柔和渐变点缀
- **清晰**：大字号，高对比，触控友好

### 6.2 色彩系统

| 角色 | 色值 | 用途 |

| 背景主色 | #F9F6F0 | 全局背景 |

| 文字辅色 | #9A9A9A | 说明文字、占位符 |

### 8.1 功能验收清单

|------|--------|----------|
| 1 | 首次启动 | 自动注入示例数据，展示课本列表 |

| 7 | 调速 | 滑块调整后，语速明显变化 |
| 8 | 重复次数 | 设置 2 次，实际朗读 2 次，间隔约 2 秒 |
